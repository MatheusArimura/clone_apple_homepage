# Clone Apple Homepage

## Sobre o projeto

Esse é um simples projeto onde tento clonar a interface da homepage da Apple utilizando apenas HTML e CSS, usei como base o vídeo da [Fernanda Kipper](https://www.youtube.com/@kipperdev).
